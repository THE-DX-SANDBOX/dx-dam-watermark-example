🔌 DAM Plugin API

Welcome to the DAM Plugin Template API server.

Available Endpoints

GET /health

Health check endpoint

POST /api/v1/process

Process asset file and extract metadata

Headers: X-API-Key: your-api-key

Body: multipart/form-data with file and callBackURL

GET /api/v1/info

Get plugin information

Documentation

  • API Explorer (Swagger UI)
  • OpenAPI Specification

Quick Test

curl -X POST http://localhost:3000/api/v1/process \
  -H "X-API-Key: your-api-key" \
  -F "file=@image.jpg" \
  -F "callBackURL=http://your-dam/callback"
